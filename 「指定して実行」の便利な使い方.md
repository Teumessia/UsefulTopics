# **指定して実行**の便利使い

　Windows用のシステムツール『ファイル名素子呈して実行』より、様々なことができる。
　ちなみにこの今回の記事においては、フォルダのことを〝ディレクトリ〟と呼んでいます。

1. ターミナル画面の呼び出し
	1. ``cmd`` と入力することでターミナル『コマンドプロンプト』が起動
	（※1-1 現在では機能強化された『PowerShell』というターミナル・エミュレータが使えます）
	2. ``PowerShell``と入力することでより便利なターミナル『PowerShell』が起動
	（※2-1 複製＆貼付やセッションごとのタブ管理に対応）
	（※2-2 シリーズ機能(?)として1-2項の**x86版**や、エディタ機能やＧＵＩ機能を強化された『PowerShell ISE』なども存在）

2. ＯＳの基本的アプリの呼び出し
　例えば、
	+ ``notepad.exe`` と入力することで、『メモ帳』アプリを起動
	+ ``Explorer.exe`` と入力することでファイルマネージャ『エクスプローラを起動』
	（※2-1 エクスプローラはWindows OS標準のファイル管理アプリです）

3. 特定の場所を開く
　マウスだけでかんたんにできる操作では開けない場所でも、コマンドを使えば開ける可能性があります
	+ ``shell:StartUp`` と入力することで、システム起動時に実行されるアプリを追加が出来るディレクトリを表示する
	（※3-a〝スタートアップ〟というシステムの起動に合わせて、自動実行されるアプリを追加するためのフォルダを開くだけです）
	+ ``shell:Programs`` と入力することで、〝スタートメニュー〟 *2-b のディレクトリを表示する
	（※3-b ざっくりいうと**スタートアップ**および**スタートメニュー**は別のものです。スタートメニューは``Windowsロゴ``キーを押したときに出てくる、最近使ったアプリやファイルが表示される小窓です）
	+ ``shell:AppsFolder`` と入力することで、これまでにインストールされているすべてのアプリケーションソフトが表示される、〝Applications〟というディレクトリを表示する
	（※3-c 『Firefox』や『Edge』などのウェブブラウザから入手＆インストールしたアプリでけではなく、アプリストア『Microsoft Store』からインストールしたアプリも、ここで全て表示させられます。しかしながら、このディレクトリにあるものを削除しないように細心の注意を払ってください。）
